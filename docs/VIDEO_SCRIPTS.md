## Recording Tips

### Equipment Needed
- Screen recording software (OBS, Camtasia, or ScreenFlow)
- Good microphone (Blue Yeti or similar)
- Quiet environment
- 1920x1080 or 1280x720 resolution

### Pre-Recording Checklist
- [ ] Clear browser cache
- [ ] Close unnecessary tabs
- [  ] Set browser zoom to 100%
- [ ] Prepare demo data
- [ ] Test audio levels
- [ ] Practice script 2-3 times

### Recording Settings
- Frame rate: 30 FPS
- Resolution: 1920x1080
- Audio: 48kHz, stereo
- Format: MP4 (H.264)

### Editing Guidelines
- Add chapter markers
- Include captions/subtitles
- Add zoom-ins for important UI elements
- Use transitions sparingly
- Background music (subtle, royalty-free)
- Export at 1080p, 6-8 Mbps

### Publishing
- YouTube (public/unlisted)
- Embed in documentation
- Add to README
- Create thumbnails
- Include timestamps in description
